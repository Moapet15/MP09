Diferències entre imatges de mapa de bits i imatges vectorials

Les imatges de mapa de bits es construeixen a partir de píxels, fet que les fa més adequades per a fotografies o elements visuals amb detalls complexos. Tanmateix, poden perdre qualitat si s’amplien excessivament.

Les imatges vectorials, com les generades amb SVG, es defineixen mitjançant equacions matemàtiques. Aquestes són escalables infinitament sense perdre qualitat i són ideals per a gràfics simples, logotips o icones.
